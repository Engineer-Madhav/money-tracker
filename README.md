# My Money Tracker – Web Technology Project

This is a personal finance tracking web application created using **HTML, CSS, and JavaScript**.  
It allows users to enter expenses under categories and sub-categories, track total money, manage savings, and generate basic reports.

## Features
- Add total money
- Add expenses with category & sub-category
- Add savings & goal planning
- View dashboard (balance, spent, saved)
- Filter expenses by category & month
- Data stored using browser localStorage

## Technologies Used
- HTML
- CSS
- JavaScript  
- localStorage API

## How to Run
1. Clone the repository or download ZIP.
2. Open `index.html` in any browser.
3. Start entering data — no installation required.
